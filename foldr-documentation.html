Foldr Documentation

Thanks for picking up Foldr — a landing page template for AI and SaaS products, built with plain HTML, CSS and JavaScript. No build tools, no frameworks, no dependencies to install.

File structure

Everything you need is in the root folder — unzip and you're ready to edit.

foldr-template/
  ├── index.html — Home 1 (light)
  ├── home-dark.html — Home 2 (dark)
  └── documentation.html — this file

Getting started

  1. Unzip the downloaded package anywhere on your computer.
  2. Open index.html directly in any browser to preview Home 1.
  3. Edit the HTML in any text editor (VS Code, Sublime, even Notepad) and refresh the browser to see changes — no compiling or build step required.
  4. When you're ready to publish, upload the files to any static host: Netlify, Vercel, GitHub Pages, or standard cPanel hosting all work.

Customizing colors

All colors are defined once, at the top of the <style> block, as CSS variables. Change a value here and it updates everywhere in the page automatically.

--paper — page background
--ink — main text / headline color
--signal — primary accent (buttons, highlights)
--peri — secondary accent (icons, small details)

To retheme the template for a different product, changing these four variables is usually enough.

Customizing fonts

Foldr uses three Google Fonts, loaded in the <head>:

RoleFontUsed for
DisplaySpace GroteskHeadlines
BodyInterParagraphs, buttons
MonoIBM Plex MonoLabels, prices, eyebrows

To swap a font, replace the Google Fonts <link> in the head and update the matching --font-* variable.

Editing content

All copy — headlines, feature text, pricing, testimonials, FAQ — is plain text inside the HTML. Search for the text you want to change and replace it directly. No CMS or database is involved.

The pricing section, testimonial cards and FAQ items are each self-contained blocks — duplicate a block and edit it to add more of the same.

Sections overview

  • Hero — headline, subheadline, two CTAs, animated sorting illustration
  • Logo strip — social proof row for client/partner logos
  • How it works — 3-step numbered process
  • Features — 4-card grid with icons
  • Pricing — 3-tier comparison with a featured plan
  • Testimonials — 3 quote cards
  • FAQ — accordion, click to expand
  • Final CTA + Footer
